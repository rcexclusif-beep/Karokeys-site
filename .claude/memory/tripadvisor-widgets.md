## Widgets écartés
- Bravo! : vert fluo incompatible avec la palette premium
- Avis élogieux : trop basique visuellement
- Bouton réseaux sociaux : hors sujet pour un site de conversion directe
- Lisez les avis : redondant avec Extraits d'avis
- Écrire un avis : écarté de confirmation.html sur décision du client
